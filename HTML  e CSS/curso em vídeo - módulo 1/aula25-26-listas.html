Listas.

São muito importantes pois:

  1. Possibilitam uma melhor leitura;
  2. Deixam o conteúdo melhor organizados;
  3. Os itens são apresentados de maneiras mais simples;

Listas ordenadas <ol>.

O exemplo de lista acima é um de uma lista ordenada, onde a ordem dos valores é importante e escrevemos assim: 

            
                <ol type="1">
                    <li>Possibilitam uma melhor leitura;</li>
                    <li>Deixam o conteúdo melhor organizados;</li>
                    <li>Os itens são apresentados de maneiras mais simples;</li>
                </ol>
Segundo a W3C, para a tag <ol> é obrigatório a tag de fechamento, mas para a tag <li> é opcional para HTML5.

O parâmetro type possui as seguintes opções de valores:

  • 1- Para criar listas numeradas (1,2,3,4...)
  • A- Para criar listas alfabéticas maiúsculas (A,B,C,D...)
  • a- Para criar listas alfabéticas minúsculas (a,b,c,d....)
  • I- Para crias listas com algarismos romanos maiúsculas (I, II, III, IV...)
  • i- Para crias listas com algarismos romanos minúsculas (i, ii, iii, iv...)

Também logo após o type, pode-se ainda adicionar o parâmetro start="#", assim, a lista inicia após seu valor (#).

Listas não ordenadas <ul>.

São listas nas quais a ordem dos elementos não é importante, e o parâmetro Type possui os seguintes valores:
disc: Uma bola preta totalmente preenchida;

  • A
  • B
  • C

circle: Uma bola com a borda preta mas sem preenchimento;
  • D
  • E
  • F

square: Pequeno quadrado totalmente preenchido.
  • G
  • H
  • I

Lista de definições <dl>

É uma lista sem demarcadores mas é bem útil. Toda a lista está dentro da tag <dl> (Definition list) onde cada termo é um <dt> (Definition term) e cada descrição é um <dd> ( Definition description ).
