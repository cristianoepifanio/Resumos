Hyperlinks.

Os Hyperlinks nos permite ligar um ponto s outro na WWW, os criamos em HTML com a tag <a> (âncora) e colocamos a URL entre as aspas do parâmetro href. 

<a href="(URL aqui)" target="#" rel="#">(âncora)</a> 

aulas <a href="https://www.youtube.com/watch?v=LeOVXQDsAIY&list=PLHz_AreHm4dkZ9-atkcmcBaMZdmLHft8n&index=27" target="_blank" rel="external">27</a>
Também podemos utilizar o atributo target que possui os seguintes valores:
  • _blank: Vai abrir o link em uma nova aba (link externo).
  • _self: Vai abrir o link na janela ou no frame atual (link interno).
Em seguida temos o parâmetro rel que indica qual é a natureza do destino, e possui os seguintes valores:
  • next que o link é a proxima parte da página.
  • prev que o link é a parte anterior da página.
  • author que é um link para o site do autor da página.
  • externala que é um link para outro site que não faz parte do site atual.
  • nofollow que é um link para um site não endossado, como um link pago.

Links para downloads.

<a href="(caminho do arquivo.)" download="(nome.formato do arquivo)" type="(media type)">(âncora)</a>
Aqui estão alguns exemplos de media type para os formatos de arquivos que mais utilizamos no nosso dia a dia:
  • application/zip
  • text/html
  • text/css
  • text/javascript
  • video/mp4
  • video/H264
  • video/JPEG
  • audio/aac
  • audio/mpeg
  • font/ttf
  • image/jpeg
  • image/png
Outros media type de outros formatos podem serem encontrados em IANA.org.
