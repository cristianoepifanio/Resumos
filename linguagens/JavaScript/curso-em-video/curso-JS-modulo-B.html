Comentário em JavaScript

São muito importantes para a organização dos nossos códigos ou para a copreensão do mesmo por um terceiro. E eles são feitos da seguinte forma:

  • // - para comentar uma única linha
  • /* */ - para comentar mais de uma linha de código.

Identificadores

Regras para a nomeação de variáveis:

  • Podem começar com letra, $ ou _;
  • Não podem começar com números;
  • É possível usar letras ou números;
  • É possível usar acentos e símbolos;
  • Não podem conter espaços;
  • Não podem ser palavras reservadas;
Obs.: - Letras maiúsculas e minúsculas fazem diferença. Tente utilizar nomes coerentes para as variáveis.

            </script>
            

            <script>

                (Se o número for racional.)
            </script>
Mas se digitarmos apenas Number (n), o JavaScript "se vira" interpretando se é um número Int ou Float

Converção número → string

String(n) ou n.toString() sendo n a variável que se quer converter.

Outra forma de concatenar é: `string ${var}` conhecida como template string, onde podemos escrever de modo mais elegante.

Sendo var s = JavaScript, os comando:

  • s.length - Diz quantos caracteres a string tem;
  • s.toUpper - Deixa toda a string em Maiúsculo;
  • s.toLowerCase - Deixa toda a string em Minúsculo;
  • document.write(`string`) - Escreve no body da página.

Formatando números

Sendo var n1 = 1545.5:

  • n1.toFixed(2) → "1545.50"
  • n1.toFixed(2).replace('.',',') → "1545,50"
  • n1.toLocaleString('pt-BR', {style: 'currency',currency:'BRL'}) → "R$ 1,545.50"

Operadores do JavaScript

Os operadores são os seguintes abaixo, cujo a ordem de precedência sucede de cima pra baixo (↓):

  • Aritmétricos - (parenteses (), potência **, multiplicação *, divisão /, resto de divisão %, soma + e subtração -) segundo a ordem de precedência (→);
  • Atribuição - (n = n + 3 ou simplesmente n += 3) por exemplo;
  • Relacionais -(>, <, >=, <=, ==, !=)
  • Lógicos - (!, &&, ll)
  • Ternário - (`string` ? true : false)
