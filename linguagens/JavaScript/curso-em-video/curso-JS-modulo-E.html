Repetições

Estrutura de repetição no início e no final

            
                while (condição desejada) {
                    //bloco de comandos
                }
            
        
            
                do {
                    //bloco de comandos
                } while (condição desejada)
Por exemplo: 
            
                var num = 1
                while (num <= 6) {
                    console.log(`Passo ${num}.`)
                    num++
                }
                        ou
                var num = 1
                do {
                    console.log(`Passo ${num}.`)
                    num++
                } while (num <= 6)

Estrutura de repetição por incremento

        
            for (início; teste lógico; incremento)  {
                //bloco de comandos
            }
          
            for (var num = 1;num <= 6;num++)  {
                console.log(`Passo ${num}.`)
            }
