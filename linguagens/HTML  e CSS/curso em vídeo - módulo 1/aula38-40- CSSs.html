Formas de estilo.

Para modificar os estilos de um site, tem-se 3 formas de faze-lo:

  1. CSS inline style: Onde a modificação é feita na linha que se quer alterar o estilo. (procure evitar)
  2. CSS internal style: Todas as modificações são feita no <head> da página (use para pequenas configurações).
  3. CSS external style: Aqui, é criado um arquivo externo ao index.html, chamado de stile.css por exemplo, onde o estilo do site é descrito nele. Este arquivo externo, possui um link no <head> de cada página na qual se queira que estilo esteja presente. (use sempre que puder)
